# 🎵 Advanced Telegram Music Bot 🎶  

A **highly advanced** and **feature-rich** Telegram music bot that plays high-quality music in voice chats, supports AI chat, lullaby mode, story narration, and much more!  

## 🚀 Features  
✅ **Perfect Audio Quality** (Like Spotify)  
✅ **Fast & Instant Commands**  
✅ **AI-Powered Chat Assistant (Casa)**  
✅ **Play Audio/Video from URLs & Files**  
✅ **Lullaby Mode for Soothing Music**  
✅ **Story Mode (Narrates Stories Based on Mood)**  
✅ **Admin Controls (/skip, /end, /pause, /resume, etc.)**  
✅ **Playlist Support & DJ Mode**  
✅ **Auto-Restart on Crash**  
✅ **Voice Command Support**  
✅ **Live Song Requests with User Voting**  
✅ **Beautiful UI & Animated Messages**  
✅ **And Many More Unique Features!**  

## 🛠 Deployment  
### Deploy on **Koyeb** or **Termux**  
1. **Clone Repository:**  
   ```bash
   git clone https://github.com/your-repository-name.git && cd your-repository-name
2. Install Dependencies:

pip install -r requirements.txt


3. Set Up Environment Variables:

API_ID

API_HASH

BOT_TOKEN

MONGO_URI

OPENAI_API_KEY
(Check .env file or manually export them in Termux/Koyeb.)



4. Run the Bot:

python3 main.py



📜 Commands

/play <song_name> – Play any song

/skip – Skip current song (Admins only)

/end – Stop music (Admins only)

/pause – Pause current song

/resume – Resume paused song

/seek <time> – Jump to a specific timestamp

/vplay – Play videos in voice chat

/pf – Force play a song

/auth <user_id> – Authorize a user for admin controls

/lyrics <song_name> – Get song lyrics

/lullaby – Play relaxing lullabies

/story <mood> – AI-generated story narration

/help – Show all commands
