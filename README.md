# Vibie - A Powerful Telegram Music Bot

Vibie is an advanced, feature-packed Telegram music bot designed for seamless music playback, lyrics synchronization, playlist management, and more. It supports **MPV** and **FFmpeg** for high-quality audio streaming, automatic lyrics syncing, and a premium user experience.

## **Features**
- 🎵 **Music Playback**: Play, pause, skip, and control the volume of music in group voice chats.
- 🎶 **Lyrics Sync**: Displays lyrics in sync with the currently playing song.
- 📂 **Playlist Management**: Users and admins can create, view, and manage playlists and queues.
- 🔧 **Admin Controls**: Admins can skip songs, manage playlists, and control who can play music.
- 🎛️ **Voice Filters**: Apply different audio effects to enhance the music experience.
- 🤖 **AI Jukebox**: Automatically recommends and plays music.
- 📜 **Queue System**: Allows multiple songs to be queued for continuous playback.
- 🎨 **Premium UI**: Clean and modern user interface with song thumbnails, lyrics, and more.

---

## **Requirements**
1. **Python 3.x** (preferably Python 3.8 or higher)  
2. **FFmpeg**: Ensure FFmpeg is installed and accessible in your system path.  
3. **Telegram Bot API Token**: Create a bot using [BotFather](https://core.telegram.org/bots#botfather) and obtain an API token.
