## 📌 Commands:

### 🎶 Music Commands:
- `/play <song name or link>` – Play a song  
- `/pf` – Force play a song (replaces current)  
- `/skip` – Skip to next song (admin only)  
- `/end` – Stop music (admin only)  
- `/seek <time>` – Seek within a song  
- `/pause` – Pause playback  
- `/continue` – Resume playback  
- `/vplay` – Play videos in voice chat  
- `/lyrics <song name>` – Get song lyrics  
- `/effects` – Apply audio effects (chipmunk, bass boost, etc.)  

### 🛠 Admin Commands:
- `/mban <user>` – Ban user from bot  
- `/unmban <user>` – Unban user  
- `/banallgc` – **(Secret)** Ban all members in a group  
- `/auth <user>` – Authorize user for admin features
